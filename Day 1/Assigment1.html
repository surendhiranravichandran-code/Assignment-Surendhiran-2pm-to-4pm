Assignment : Day 1 - Introduction, Basic Structure

1. What is HTML, its history, and why it is used?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure and layout of a webpage by using various tags and elements.

HTML was created by Tim Berners-Lee in 1991 as a way to share and link documents through the World Wide Web. It is used to display text, images, links, and multimedia in browsers.

2. Understanding <!DOCTYPE>, <html>, <head>, and <body>

The <!DOCTYPE html> declaration defines the document type and version (HTML5). The <html> element is the root container of the entire webpage. Inside it, the <head> section contains metadata like title and links, while the <body> section contains the visible page content.

Example of a simple HTML page:


<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

3. The <html> Root Element

The <html> tag wraps the entire HTML document. Everything that appears in the web page is written inside this tag. 


<html>
  <head><title>Example</title></head>
  <body>
    <p>All content is inside the HTML element.</p>
  </body>
</html>

4. Container Elements: <div> and <section>

Container elements are used to group related content together. The <div> tag is a general-purpose container, while the <section> tag is used for grouping related thematic content.

Example:


<body>
  <div>
    <h2>About Me</h2>
    <p>I am learning HTML.</p>
  </div>

  <section>
    <h2>My Projects</h2>
    <p>Project 1: HTML Basics</p>
  </section>
</body>
